## 🎯 Philosophie pédagogique

> *"Les ressources que je crée doivent fonctionner sans connexion, sans compte, sans friction — directement dans le navigateur de l'élève."*

- **Autonomie** : chaque fichier HTML est une application complète et standalone
- **Qualité** : audit systématique avec scores numériques, itérations jusqu'à un niveau professionnel
- **Modernité** : APIs récentes (PyScript 2026, ES2020+, CSS custom properties), pas de jQuery en 2026
- **Accessibilité** : ARIA, contraste, navigation clavier — l'inclusion est non-négociable
- **Open** : tout est libre, réutilisable, modifiable — CC BY-SA ou domaine public
