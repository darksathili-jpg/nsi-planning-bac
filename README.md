## Principes de conception

Toutes les ressources respectent les mêmes contraintes :

- **Standalone** — un seul fichier `.html` ou `.md`, zéro dépendance réseau obligatoire
- **Modernes** — APIs récentes (PyScript 2026.3.1, ES2020+, CSS custom properties)
- **Accessibles** — attributs ARIA, contraste WCAG, navigation clavier
- **Itérées** — chaque ressource passe par un cycle d'audit avec score avant publication
- **Libres** — réutilisables, modifiables, sans restriction
